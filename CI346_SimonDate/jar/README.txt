----------------------------
How to run java on localhost
----------------------------

Server- Click on the icon. Server wil run in background until killed through task manager.

Player- Click on icon. You will be matched as a pair with the next player to connect

observer- open command line. open the directory that the jar is stored on 
(for desktop it would be C:/Desktop/CI346_SimonDate/jar) and run:

java -jar observer.jar 0  

Will join lobby 0. Change the number at the end to observer the corresponding lobby.
You can also run the Server and Player jars using the command line too.

There is another README. In the src folder for eclipse running.
